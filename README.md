# Hi, I'm Ezequiel 👋

I'm a PhD candidate in ML and AI methods for astronomical imaging at the [CosmoStat laboratory](https://www.cosmostat.org/), CEA-Saclay, Paris.

## 🔭 Research Interests

- PSF modelling for space-based optical telescopes
- Radio interferometry modelling and simulation
- Forward modelling and generative modelling
- Weak gravitational lensing, cosmic shear inference

## 🚀 Main Projects

| Project | Description |
|---|---|
| **[SHINE](https://github.com/CosmoStat/SHINE)** | A JAX-powered framework for probabilistic shear estimation in weak gravitational lensing |
| **[argosim](https://github.com/ARGOS-telescope/argosim)** | A lightweight, modular JAX-powered package for simulating radio interferometric observations |
| **[wf-psf](https://github.com/CosmoStat/wf-psf)** | A TensorFlow package for data-driven modelling of the instrumental response of optical telescopes |
| **[Radio-WL](https://github.com/CentofantiEze/Radio_WL_Generative_Forward_Model)** | A differentiable generative forward model and Bayesian shear inference pipeline for radio weak lensing |
| **[Spectral-class](https://github.com/CentofantiEze/sed_spectral_classification)** | PSF-aware spectral star classifier from single wide-band images |
| **[argosim-GUI](https://github.com/CentofantiEze/argosim-GUI)** | A graphical user interface for the argosim package, mainly for educational purposes |
